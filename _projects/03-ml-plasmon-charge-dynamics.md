### Project description

Developed a **physics-informed machine learning** time-series model using **PyTorch + GNNs** to predict **electron trajectories** in metal nanoparticles.

### Highlights

- Delivered **~10,000× speedup** compared to real-time TDDFT-style approaches.
- Published: *Machine learning models capture plasmon dynamics in Ag nanoparticles* (J. Phys. Chem. A, 2023).

### Technologies used

- PyTorch (training + inference)
- Graph neural representations for nanoparticle geometry
- Physics-informed constraints / inductive biases for stable rollout
