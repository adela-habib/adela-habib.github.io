### Description
Developed a robust graph-neural-network machine learning force field for ionic liquids using active-learning sampling and ensemble uncertainty to target new configurations.

### Scale
Used the resulting ML-FF to run molecular dynamics on complex ionic liquids at very large scale (up to million-atom regimes).
